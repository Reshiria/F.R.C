This is the branch which serves to form the code documentation. It uses the access point of the [[Introduction]].

I'm personally organizing code into a few different sections(although I'd highly recommend Austin follows similar procedure when making his docs).

Code 1XX will be the very foundational basics, that is, what is code, how does one code.
Code 2XX will be the beginning of FRC coding, what the different loops are and what they do. Once you're through here, you should know how to code a basic robot
Code 3XX will be more advanced processes which end in better results. PID, trapezoid profiles, subsystem differentiation, logging.
Code 4XX will be the highest level of code. It's the highest bc it's the most I know how to do and nothing exists above me bc I am totally the best. This level consists of things like auto path following, tuned vision, design advice for systems, and automatic driving.

While you don't need to read it all, I'd highly recommend at least scanning through it.

In addition, there will be a tldr at the very bottom. 

All of these resources can be found elsewhere on the wide web, but I thought it might be helpful to have them concisely(even if a bit hard to understand) bunched in one place.
