Before beginning this section in full, there's a few things that need to happen. Namely, setting up the project. Download your year's WPIlib from the [FIRST website](https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html). WPIlib is the handy dandy coding library built specifically for FIRST programs by the Worchester Polytechnic Institute.

Along with WPIlib, get the game tools. These are currently by NI but I believe they'll change in the future. Actually, some of this won't even be applicable since rios are changing next year.

Once you've downloaded the folder, unzip and run it. It should prompt you with the proper downloader. Choose to download for this computer.

Once you're done, open up your years VS code.
In the top right, there will be a unique symbol next to the run arrow. This is your gateway into all things frc project. Click onto it and choose to make a new project.

For project type, select Template>>java>>Command Robot

Find your repository folder and set the base location to be there

Give your project a name that's clear and indicative of what it is.

Finally, fill in your team number and click generate project.

Your project should now be complete.
